This script for termux extra key
and you can do install keys in your termux

## INSTALLATION :

* apt-get update
* apt-get upgrade
* pkg install git
* git clone https://github.com/losser-anonymous/TERMUX-KEYS
 * && chmod +x * &&
* bash TERMUX-KEYS.sh

And guys you want uninstall this key so type

cd TERMUX-KEYS &&
bash remove-key.sh
